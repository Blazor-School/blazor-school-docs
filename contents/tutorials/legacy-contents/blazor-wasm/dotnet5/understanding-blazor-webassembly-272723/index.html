Understanding Blazor WebAssembly

Most people will find it too confusing when starting a new technology, that is why we created this series. This guide series will help you become a proficient Blazor WebAssembly programmer. To understand Blazor WebAssembly, you need to learn the following building blocks:

  • Components: The main building block for Blazor WebAssembly websites.
  • Directives: Define the behaviour of an element in Blazor WebAssembly websites.
  • Dependency Injection: Classes that perform a specific function and help the component decoupled.
