Introduction to the Blazor Server Docs

Blazor Server is a free open source framework developed by Microsoft to help create single-page app websites development in C# and HTML.

These Blazor Server Docs help you learn and use the Blazor Server framework and development platform, from a very simple application to optimizing complex contexts for enterprises. Tutorials and guides include downloadable examples to help you learn easier.
